# Changelog - Vitracka

Histórico de alterações e melhorias realizadas no projeto.

---

# v0.4 Alpha

**Status:** Em desenvolvimento

## Novidades

### Integração com modelo de IA via Hugging Face

* Substituição do modelo de IA local por uma solução baseada em modelos disponibilizados pela plataforma Hugging Face.
* Modelo utilizado: **openai/gpt-oss-20b**
* Atualização da arquitetura de processamento de respostas do chatbot.
* Preparação da arquitetura para futuras evoluções relacionadas à inteligência artificial.

### Novas páginas institucionais

Foram adicionadas novas páginas ao sistema:

* Página "Sobre";
* Página "Política e Privacidade";
* Página "Funcionalidades";
* Página "Começar".

Essas páginas têm como objetivo melhorar a apresentação do projeto, fornecer informações aos usuários e facilitar a navegação inicial.

---

## Melhorias

### Interface do sistema

* Reformulação visual de componentes da aplicação;
* Melhorias na organização dos elementos da interface;
* Ajustes para proporcionar uma experiência de usuário mais consistente.

### Padronização da navbar

* Implementação de uma estrutura mais padronizada para a barra de navegação;
* Ajustes visando manter uma identidade visual consistente entre as páginas;
* Melhor organização dos elementos de navegação.

### Otimização do chatbot

* Melhorias no comportamento visual do chatbot;
* Implementação de animações para tornar a interação mais dinâmica;
* Adição de efeitos visuais utilizando AJAX;
* Melhorias na experiência durante o envio e recebimento de mensagens.

---

## Observações

A versão **v0.4 Alpha** representa uma etapa significativa no desenvolvimento do Vitracka, com foco na evolução da inteligência artificial, melhoria da experiência do usuário e estruturação da interface do sistema.

Por se tratar de uma versão Alpha, podem existir instabilidades e comportamentos inesperados que serão tratados em versões futuras.
